# コントリビュート

このページは開発中です。

## ライブラリ

[online-judge-tools/verification-helper](https://github.com/online-judge-tools/verification-helper) を使ってテストしています。

### インストール

```
pip3 install online-judge-verify-helper
```

### 実行

```
oj-verify run
```

初回は時間がかかります。

## ドキュメント
